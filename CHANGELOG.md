# Changelog

All notable changes to CrewTrack will be documented in this file.

## [1.0.0] - 2026-07-12

### Added
- RFID card scanning with RC522 reader
- ST7789V 2" LCD display with multiple screens
- MicroSD card attendance logging
- Buzzer audio feedback
- WiFi Access Point with web dashboard
- Employee management (add/edit/delete)
- Duplicate scan detection
- Monthly salary calculation
- CSV export
- Search and filter workers
- NTP time synchronization
- Responsive dark-theme dashboard

### Hardware
- ESP32 DevKit V1
- MFRC522 RFID reader
- ST7789V 2" IPS LCD
- MicroSD card module
- Active buzzer
